# 玩家说明 — 侠客行 Web 版

## 这是什么

轻量文字武侠 RPG：场景描写 + 按钮操作 + 见闻日志。无需输入 MUD 命令。

## 基本操作

- **走动**：点出口 → 预览 →「前往」
- **环顾**：刷新当前房间描写
- **修炼**：打坐 / 吐纳 / 练功；可设停止条件后挂机
- **动手**：选目标战斗；可开「自动普攻」与残血应对
- **角色**：点顶栏名字 → 仪容 / 气血 / 档案 / 武功 / 行囊
- **地图**：区域（扬州）与世界地标

## 挂机助手（无脚本）

Web 版**不支持**玩家编写机器人或触发器。

官方提供可配置助手：

| 功能 | 说明 |
|------|------|
| 修炼挂机 | 选方式 + 停止条件（满/次数/潜能尽）；遇战可自动停 |
| 战斗辅助 | 自动普攻；气血低于 X% 时提示 / 逃跑 / 停手 |

随时点「停止」或「停手」手动接管。

## 与传统 Telnet MUD 的差异

- 无命令行；界面为按钮与面板
- 首发区以扬州及周边练级线为主，非全图保证
- 结构化数据由服务端推送，体验更稳定
