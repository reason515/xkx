# 首发区范围（阶段 0.5）

Web 版首发不覆盖全图约 6400 间房，仅保证以下区域可稳定游玩。

## 核心：扬州城

| 区域 | 路径前缀 | 说明 |
|------|----------|------|
| 客店与周边 | `d/city/kedian.c` 及相邻 | 新手出生点、5 步引导 |
| 北大街 / 南大街 | `d/city/*dajie*` | 城内主街 |
| 驿站、马厩、花店 | `d/city/` | 客店出口可达 |
| 当铺、铁匠、药铺 | `d/city/` | 基础经济 |
| 武庙、擂台 | `d/city/wumiao.c`, `leitai.c` | 修炼与切磋 |

## 练级线（1～2 条）

| 线路 | 路径 | 等级建议 |
|------|------|----------|
| 城南树林 | `d/city/seforest.c` → `d/forest/` | 新手～10 级 |
| 北门郊外 | `d/city/neforest.c` 及延伸 | 10～20 级 |

## 暂不保证

- 门派拜师与长途任务链
- 全地图传送与所有世界地标点击可达
- 巫师命令与 admin 工具

## 地图 UI

- **区域** Tab：扬州 5×5 格子 + 楼层（客店雅房等）
- **世界** Tab：参照 `doc/help/map_all` 地标；仅扬州默认可交互，其余为预览
